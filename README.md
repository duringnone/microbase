# microbase
common function /method for GoMicro-Beego

```
    └─microbase           // 共用方法
    |    ├─base.go          // 环境变量,全局变量初始化

    |    | 
    |    ├─service.go		// 字符串/数组/类型转换/...处理方法
    |    |                     
    |    ├─third.go 		// 第三方服务 [邮件发送服务]
    |    │
    |    └─validate.go      // 验证器
    | 	
    └─...
   


```
